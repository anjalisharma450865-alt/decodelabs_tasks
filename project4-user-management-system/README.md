# 🚀 User Management System

A full-stack web application to manage users using Node.js, Express, MongoDB, and JavaScript.

## 🔥 Features
- Add user
- View users
- Edit user
- Delete user
- Real-time UI updates

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB Atlas

## 🚀 How to Run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
Open index.html in browser  

## 🌐 API Endpoints
- GET /api/users  
- POST /api/users  
- PUT /api/users/:id  
- DELETE /api/users/:id
